Mes Compétences

Je connais HTML depuis 2017.

Je connais CSS depuis 2017.

Je connais JavaScript depuis 2017. Je m'améliore un peu plus chaque mois tout au long de la formaton !

Découvert le 11 Novembre 2022, je débute dans ce langage de programmation. Je m'intéresse plus particulièrement à SCSS que SASS.

Typescript est pour moi l'outil indispensable d'un développeur car, il aide beaucoup dans la résolution de problèmes et permet d'éviter les erreurs pour coder correctement.

J'ai pu découvrir wordpress lors de ma formation front-end. Nous avons découvert quelques extensions dont wooCommerce, mais également les thèmes Divi & élémentor.

React est vraiment bien pour l'organisation ainsi que pour la création d'un gros projet.

Visual Studio Code est le meilleur logiciel pour développer des sites internet !

J'utilise Canva pour créer des visuels sur les réseaux sociaux.

J'utilisais photoshop pour créer des contenus sur les réseaux sociaux, mais également pour créer des miniatures pour mes vidéos YouTube.

Utilisé depuis 2018 pour monter mes vidéos pour ma chaine YouTube, Adobe Première Pro est pour moi le meilleur logiciel de montage vidéo !
